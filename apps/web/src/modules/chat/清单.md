# 智能助手

## 已经完成的功能

- 对话交互 流式输出
- 会话管理，可切换会话，确保上下文一致性
- 视觉与体验 Gemini 风格视觉
- 对话持久化，IndexDB存储，确保刷新不丢失

## 需求清单

1. 会话管理

- 多会话支持：左侧侧边栏显示历史对话列表，点击可切换。
- 新建会话：一键开启全新对话上下文。
- 会话操作：支持对历史会话进行删除。
- 持久化存储：对话记录应保存到本地 IndexedDB，保证刷新不丢失。

2. 对话交互

- 流式输出 (Streaming)：这是 Gemini 的核心体验，文字逐字跳出，而不是等待全部生成后才显示。
- Markdown 深度支持：
- 代码高亮：支持主流编程语言的语法高亮和一键复制。
- 表格与列表：规范的排版。
- 数学公式 (LaTeX)：支持科学计算和公式显示。
- 停止生成：在 AI 输出过程中，用户可以随时中断生成。
- 重新生成 (Regenerate)：对上一次的问题要求 AI 给出不同的回答。
- 快捷操作：每条消息下方提供“复制内容”、“点赞/踩”等反馈按钮。

3. 多模态与输入

- 图片上传/分析：利用 Gemini 的多模态能力，支持拖入或粘贴图片并进行对话。
- 多行输入框：支持 Shift+Enter 换行，动态调整输入框高度。
- 快捷指令 (Slash Commands)：如输入 / 触发预设的角色模板或指令。
- 空白页引导：在没有消息时，显示类似 Gemini 的建议提示词卡片（例如：“帮我写一段 React 代码”、“解释量子力学”）。

4. 视觉与体验

- Gemini 风格视觉：
  简洁的现代感设计，使用柔和的阴影和圆角（如你现在的 rounded-[2.5rem]）。
  精致的动效（Loading 状态、消息进入动画）。
- 响应式布局：完美适配移动端和桌面端。
